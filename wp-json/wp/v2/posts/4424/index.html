Fork and Cream Sauce<\/strong> es un curioso accesorio<\/a><\/strong> con el cual puedes enrollar tus cables en un tenedor para mantener el orden.
\n
\nComo pod\u00e9is ver en la imagen es un tenedor que va encajado en una pieza que act\u00faa como base y el cable va enrollado<\/strong> como si de un espagueti se tratase. Esta idea viene para intentar mantener un poco el orden de cables que tenemos con nuestros dispositivos.<\/p>\n

Ellos mismos afirman que este invento les viene del plato de espaguetis revuelto que adquiere un orden cuando se enrollan en un tenedor. Este accesorio<\/strong> est\u00e1 disponible para todos los cables que quepan, pero afirman que el cable del iPhone o similar es el que mejor le va,<\/strong> queda est\u00e9ticamente muy bien y mantiene la filosof\u00eda del Mac<\/a><\/strong> de tener cuantos menos cables mejor. En este caso el cable no desaparece, pero si que se reduce y queda ordenado.<\/p>\n

El punto positivo de este producto es que la empresa que lo produce est\u00e1 vinculada con Save the Children<\/strong> y la gracia del asunto es que si compras un tenedor, Save the Children ser\u00e1 capaz de ofrecer una comida a un ni\u00f1o hambriento.<\/strong><\/p>\n

El env\u00edo es a todo el mundo y tiene un precio de 13 d\u00f3lares americanos con 4 d\u00f3lares americanos<\/strong> de env\u00edo. El accesorio se enviar\u00e1 3 d\u00edas despu\u00e9s de recibir el pago.<\/strong> El m\u00e9todo de pago es v\u00eda PayPal<\/strong>.<\/p>\n
