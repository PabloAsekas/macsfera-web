Apple<\/a> ha confirmado que ha sido v\u00edctima de un ataque organizado inform\u00e1tico que ha afectado a varios equipos pero que no han conseguido robar informaci\u00f3n.<\/strong><\/p>\n

<\/p>\n

Los hackers que llevaron a cabo el ataque usaron una vulnerabilidad de Java<\/strong> que afect\u00f3 a varios ordenadores en la compa\u00f1\u00eda. Estos fueron descubiertos a tiempo y aislados de la red. No hay evidencias de que les hayan robado ninguna informaci\u00f3n.<\/strong><\/p>\n

La vulnerabilidad se encuentra en un plugin de Java para navegadores, otra empresa v\u00edctima de este ataque fue Facebook la semana pasada. Apple trabaja con la polic\u00eda para ver si encuentran el foco del ataque y poder tomar medidas.<\/strong><\/p>\n

La manzanera, consciente de estas posibles vulnerabilidades, entrega los Macs<\/strong> sin Java instalado, adem\u00e1s, para los que lo tenemos instalado, si no lo usamos durante 35 d\u00edas se deshabilita autom\u00e1ticamente. No obstante, la empresa ha lanzado una actualizaci\u00f3n menor que implementa la seguridad y protege contra este tipo de ataques.<\/strong><\/p>\n

Al parecer este ataque empez\u00f3 y se extendi\u00f3 a trav\u00e9s de un sitio web para desarrolladores de software, son varias las webs o servicios que han sufrido ataques muy parecidos y distribuidos por sitios similares.<\/p>\n
