Algunos cambios fueron los de a\u00f1adir nuevas sesiones m\u00e1s cortas de media hora<\/strong> y el cambio del nombre \u00abTalleres\u00bb por \u00abSesiones de Grupo\u00bb para hacer las clases con otros usuarios.
