\nLa pegatina ocupa toda la superficie t\u00e1ctil del Magic Trackpad<\/strong> y en algunas zonas tendr\u00e1 impresa los distintos n\u00fameros. Magic Numpad<\/strong> te dar\u00e1 tres posibles extensiones distintas:<\/p>\n

  • Teclado num\u00e9rico con espacio<\/strong>: La pegatina con las teclas num\u00e9ricas est\u00e1 a la derecha y a la izquierda queda una superficie para poder usar el trackpad como rat\u00f3n.<\/li>\n

    El contenido del paquete es de las tres pegatinas, un boli permanente<\/strong> para asignar la funci\u00f3n de las teclas personalizables, una tarjeta con los datos de la aplicaci\u00f3n,<\/strong> un trapo<\/strong> para limpiar el trakpad y una tarjeta que nos ayudar\u00e1 a fijar las pegatinas.<\/strong><\/p>\n
