AirDrop<\/h2>\n

Si bien iCloud nos sirve para sincronizar archivos entre nuestros dispositivos, \u00bfqu\u00e9 hacemos si es otro usuario el que quiere pasarnos un documento? Con AirPlay podremos reproducir los v\u00eddeos, m\u00fasica, pases de fotos o presentaciones que tengamos en el Mac con cualquier pantalla externa<\/strong> siempre y cuando haya una conexi\u00f3n Wi-Fi y un dispositivo AirPlay conectado a dicha pantalla. <\/p>\n

Si bien hace mucho que no se usan los CDs o DVDs para instalar aplicaciones, Apple desarroll\u00f3 la Mac App Store<\/a> para que ni si quiera hubiese que plantearselo. Con ella podremos instalar o desinstalar cualquier aplicaci\u00f3n disponible a golpe de click.<\/strong><\/p>\n

Wi-Fi<\/h2>\n

<\/a><\/p>\n

Por supuesto, la antena Wi-Fi es un instrumento que lleva muchos a\u00f1os presente entre nosotros, y los fabricantes cada d\u00eda encuentran m\u00e1s usos para que podamos aprovechar este tipo de conexi\u00f3n.
