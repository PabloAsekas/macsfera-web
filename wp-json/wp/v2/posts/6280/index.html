QuickTime<\/h2>\n

<\/a><\/p>\n

  4. Hacemos clic en Exportar<\/strong>.<\/li>\n<\/ol>\n

    El archivo resultante tendr\u00e1 un formato .m4a o MPEG4.<\/strong><\/p>\n
