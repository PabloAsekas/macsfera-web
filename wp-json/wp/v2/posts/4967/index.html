Actualmente Xcode<\/strong> est\u00e1 disponible de forma gratuita en la Mac App Store<\/strong> pero solo pueden descargarlo e instalarlo aquellas personas que tuvieran Lion<\/a><\/strong>. Si bien porque tienes un Mac<\/strong> que no es compatible con el nuevo OS X<\/a><\/strong> o que te parece que es mejor la versi\u00f3n actual, en este tutorial descubrir\u00e1s como instalar la app para desarrollar en Snow Leopard.<\/strong>
\n
\nLos pasos a seguir son para enga\u00f1ar al sistema y aparentar tener Lion<\/strong> para que no nos de error en la descarga. Para eso tenemos que hacer lo siguiente:<\/p>\n

    \n
  1. Abrir una ventana de Finder<\/strong> y buscar el archivo SystemVersion.plist<\/strong> de la siguiente ruta: System\/Library\/CoreServices\/SystemVersion.plist<\/strong><\/li>\n
  2. Abrimos el documento que contiene unos caracteres, tendremos que localizar el n\u00famero de nuestra versi\u00f3n,<\/strong> es ah\u00ed cuando lo vamos a cambiar. Debemos sustituir lo que ponga por 10.7 pero quedaos con qu\u00e9 pon\u00eda antes.<\/strong><\/li>\n
  3. Cuando lo hayamos cambiado tan solo tendremos que guardar el archivo.<\/li>\n
  4. Ejecutamos el Terminal y ponemos el siguiente c\u00f3digo para que surja efecto:
    \n

    killall Finder<\/p><\/blockquote>\n<\/li>\n

  5. Acudimos a la Mac App Store<\/strong> y elegimos descargar Xcode<\/strong>.<\/li>\n
  6. Una vez descargado vamos a volver todo a su estado normal.<\/strong> Repetimos el proceso 1, 2 y 3 pero en vez de poner 10.7 ponemos lo que pon\u00eda antes de cambiarlo.<\/strong><\/li>\n<\/ol>\n

    <\/a><\/p>\n

    De ese modo conseguiremos instalar el Xcode<\/strong> y luego no alterar m\u00e1s el sistema, adem\u00e1s tendremos el programa instalado como una aplicaci\u00f3n<\/strong> m\u00e1s en vez de bajarte todos los archivos desde la web oficial.<\/p>\n
