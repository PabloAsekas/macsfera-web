Apple<\/strong> ha publicado una nueva herramienta que permitir\u00e1 eliminar todo rastro del malware Flashback<\/a> en cualquier sistema que no tenga Java.<\/strong><\/p>\n

<\/p>\n

Hace dos d\u00edas lanzaron una actualizaci\u00f3n de seguridad para eliminarlo en sistemas con Java<\/a><\/strong> pero ahora, puesto que Lion<\/a><\/strong> carece de Java<\/strong> de forma predeterminada, disponemos de esta herramienta para eliminar sin problema Flashback.<\/strong><\/p>\n

Esta herramienta elimina el software malicioso Flashback y sus variantes m\u00e1s comunes.<\/p>\n

Si el software malicioso Flashback se encuentra en el disco, un cuadro de di\u00e1logo le notificar\u00e1 que el malware ha sido eliminado.<\/p>\n

En algunos casos, la herramienta de eliminaci\u00f3n de Flashback deber\u00e1 reiniciar el equipo con el fin de eliminar por completo el malware Flashback.<\/p>\n

Esta actualizaci\u00f3n est\u00e1 recomendada para todos los usuarios de OS X Lion sin Java instalado.<\/p><\/blockquote>\n

La descarga se har\u00e1 desde la web de Apple<\/strong> descargando un archivo .dmg que requiere de permisos de administrador para instalarlo.<\/strong><\/p>\n
