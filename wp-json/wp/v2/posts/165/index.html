Expos\u00e9<\/h2>\n

<\/a><\/p>\n

Es una de las mejores formas de interactuar, de distinta manera, con todas las ventanas que tenemos abiertas en nuestro Mac<\/strong>. Con un movimiento del trackpad<\/strong> (en los port\u00e1tiles) o la ejecuci\u00f3n de un comando, podemos saber que ventanas tenemos abiertas, distingui\u00e9ndolas de las minimizadas, podemos interactuar por ellas de una manera r\u00e1pida. De la misma manera podemos despejar todas las ventanas y ense\u00f1ar el el escritorio o incluso, estando sobre una aplicaci\u00f3n<\/strong>, nos dice cuantas ventanas de \u00e9sta tenemos abiertas. Ocho acciones que pueden tener mucha utilidad: Todas las ventanas, Ventanas de aplicaciones<\/strong>, Escritorio, Dashboard<\/strong>, Spaces<\/strong>, Iniciar salvapantallas, desactivar salvapantallas y poner la pantalla en reposo.<\/p>\n

Spaces<\/h2>\n

<\/a><\/p>\n

Es una magn\u00edfica funci\u00f3n que tiene OS X Snow Leopard<\/a><\/strong> y es la de poder crear distintos entornos o escritorios donde poder trabajar. Esto es, al igual que en Linux podemos distribuir todas nuestras ventanas entre distintos escritorios para estar m\u00e1s ordenados y para estar menos agobiados. El sistema es capaz de emular 16 escritorios y podemos interactuar entre ellos de tres maneras distintas:<\/p>\n

    \n
  • Ejecutando el comando de teclas cmd+flechas de direcci\u00f3n:<\/strong> Esta opci\u00f3n nos permite navegar entre todos los entornos de una forma m\u00e1s lenta.<\/li>\n
  • Ejecutando el comando de teclas cmd+N\u00daMERO:<\/strong> Es la misma opci\u00f3n que la anterior pero de esta manera vas al escritorio de una forma m\u00e1s acelerada.<\/li>\n
  • Ejecutando un comando de fn+FX<\/strong> (donde X es el n\u00famero asignado en Preferencias del Sistema<\/strong>): Con este atajo de teclado obtenemos una vista general de todos los escritorios que tenemos activados, con una peque\u00f1a previsualizaci\u00f3n de las ventanas que tenemos abiertas.<\/li>\n<\/ul>\n
