\n
\nEste adaptador viene sobre todo para apaliar las posibles disputas que surjan en un futuro por la ley que se pretende aprobar en Europa de estandarizar un mismo conector<\/strong> para todos los cargadores de los m\u00f3viles y as\u00ed prevenir antes de lanzar un posible iPhone.<\/p>\n

El Adaptador de iPhone Micro USB permite el uso de terceros micro cables USB y cargadores para sincronizar o cargar tu iPhone. Basta con conectar el iPhone al adaptador de Micro USB, conecte un cable micro USB o cargador para el adaptador USB Micro.<\/p><\/blockquote>\n

El modelo estandar que la comisi\u00f3n tiene en mente es el Micro USB<\/strong> y este es, tal vez, el mejor modo que ha encontrado Apple<\/strong> para hacer compatible su sistema de 30 pines con el de micro USB<\/strong>.
