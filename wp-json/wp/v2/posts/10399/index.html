Con la presentaci\u00f3n del nuevo MacBook<\/a>, Apple introdujo una serie de accesorios oficiales para poder aprovechar toda la funcionalidad del nuevo puerto USB tipo C.<\/strong><\/p>\n

<\/p>\n

Adaptador multipuerto de USB-C a AV digital<\/h2>\n

Este adaptador se conecta al puerto USB de nuestro Mac y nos provee de otra toma USB de tipo C, un puerto HDMI para conectar el Mac a una pantalla y un puerto USB 3.1<\/strong> para poder transferir cualquier archivo entre dispositivos compatibles. A pesar de que el puerto USB-C se puede utilizar para transferir archivos tambi\u00e9n, no queda claro si en este caso podremos usarlo para ese fin, Apple s\u00f3lo especifica que lo podremos usar para cargar el Mac. El adaptador tiene un precio de 89\u20ac<\/a>.<\/p>\n

Adaptador multipuerto de USB-C a VGA<\/h2>\n

Este adaptador es exactamente igual que el anterior, pero en vez de contar con un puerto HDMI, cuenta con un puerto VGA<\/strong> de salida de hasta 1080p HD. El precio tambi\u00e9n asciende a 89\u20ac<\/a>.<\/p>\n

<\/a><\/p>\n

Adaptador de USB-C a USB<\/h2>\n

Este adaptador nos sirve para usar el puerto USB de tipo C del MacBook con el fin de poder usar un puerto USB 3.1 de cualquier dispositivo compatible.<\/strong> El adaptador de USB-C a USB tiene un precio de 19\u20ac<\/a>.<\/p>\n

Cable de carga USB-C de 2 metros<\/h2>\n

Es un cable de 2 metros de longitud, en ambos extremos contamos con un puerto USB de tipo C. Con este cable podremos cargar el Mac y transferir datos entre dispositivos<\/strong> siempre y cuando dispongan de un puerto USB-C. El cable tiene un precio de 35 euros<\/a>.<\/p>\n

Adaptador de corriente USB-C de 29W<\/h2>\n

El adaptador de corriente nos sirve para cargar el MacBook.<\/strong> Tiene el mismo dise\u00f1o que el de un iPad salvo por el puerto USB, ahora tiene una toma de USB-C en vez del corriente \u2013en este caso es necesario un cable como el antes mencionado\u2013 y que este tiene mucha m\u00e1s potencia. El cargador tiene un precio de 55\u20ac<\/a>.<\/p>\n
